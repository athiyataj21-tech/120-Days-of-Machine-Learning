# Day 1 Quick Notes

## Variables

* Used to store values in memory.
* Created using the assignment operator (=).

Example:

```python
name = "Athiya"
```

## Data Types

| Data Type | Example  |
| --------- | -------- |
| String    | "Athiya" |
| Integer   | 24       |
| Float     | 8.4      |
| Boolean   | True     |

## Useful Functions

```python
type()
int()
float()
str()
bool()
```

## Key Points

* Python is dynamically typed.
* No need to declare variable types explicitly.
* Variables can change values during execution.
* Data types determine the operations that can be performed on data.
